# منصة إدارة الإشتراكات (Subman) 🚀

منصة احترافية متكاملة لإدارة اشتراكات الخدمات البرمجية والذكاء الاصطناعي (مثل Grok, ChatGPT, Perplexity) مصممة خصيصاً لتلبية احتياجات أصحاب الأعمال والموزعين.

## 🌟 المميزات الرئيسية

### 1. لوحة البيانات والتحليلات (Dashboard)
*   **رسوم بيانية تفاعلية:** عرض توزيع المشتركين حسب الخدمة (Pie Chart) وتوزيع الدخل (Bar Chart).
*   **فلترة مخصصة:** إمكانية عرض إحصائيات الدخل والنمو لأي فترة زمنية محددة (من - إلى).
*   **مقاييس الأداء:** عرض فوري للدخل الإجمالي، عدد الاشتراكات الجديدة، وإجمالي الحسابات النشطة.

### 2. إدارة المشتركين الاحترافية
*   **نظام CRUD كامل:** إضافة، تعديل، وحذف بيانات المشتركين بكل سهولة.
*   **بيانات تفصيلية:** تشمل الاسم، البريد الإلكتروني، رقم الواتساب، رابط فيسبوك، ومساحة العمل (Workspace).
*   **بحث ذكي:** محرك بحث سريع يبحث بالاسم أو البريد أو رقم الهاتف.
*   **فلتر التجديد:** زر مخصص لعرض الحسابات التي انتهت أو التي تقترب من موعد التجديد فقط.

### 3. مركز التنبيهات الذكي
*   **تنبيهات تلقائية:** إخطارات فورية للحسابات المنتهية أو التي ستنتهي خلال يومين.
*   **نظام أرشفة:** يتم الاحتفاظ بالتنبيهات لمدة **أسبوع واحد** فقط ثم تُحذف تلقائياً للحفاظ على نظافة النظام.
*   **شارة تنبيه:** أيقونة الجرس في القائمة الجانبية توضح عدد التنبيهات الجديدة.

### 4. إدارة الصلاحيات والمستخدمين
*   **تعدد المستخدمين:** إمكانية إضافة أعضاء فريق عمل بكلمات مرور خاصة.
*   **أدوار مخصصة:** توزيع الصلاحيات بين (مدير - Admin) للتحكم الكامل، و (محرر - Editor) للعرض والتعديل فقط.

### 5. تصدير البيانات
*   **تصدير Excel:** زر مخصص لتصدير كافة بيانات المشتركين بضغطة واحدة إلى ملف `.xlsx` احترافي.

### 6. تجربة مستخدم فائقة (UX/UI)
*   **واجهة عربية بالكامل:** تصميم RTL متوافق تماماً مع اللغة العربية.
*   **استجابة كاملة (Responsive):**
    *   **على الكمبيوتر:** قائمة جانبية ثابتة وتوزيع مريح للبيانات.
    *   **على الجوال:** شريط تنقل سفلي (Bottom Nav) لسهولة الاستخدام بالإبهام.
*   **قاعدة بيانات محلية:** استخدام **IndexedDB** لضمان سرعة الأداء وحفظ البيانات حتى عند انقطاع الإنترنت.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

*   **React + TypeScript:** لبناء واجهة برمجية قوية ومنظمة.
*   **Vite:** لضمان سرعة التطوير وبناء المشروع.
*   **Dexie.js:** لإدارة قاعدة البيانات المحلية (IndexedDB) بكفاءة عالية.
*   **Recharts:** للرسوم البيانية التفاعلية.
*   **XLSX:** لتوليد ملفات الإكسل.
*   **Vanilla CSS:** لتصميم واجهة مستجيبة وخفيفة الوزن.

## 🚀 التشغيل والتطوير

### المتطلبات
*   Node.js (الإصدار 18 أو أحدث)
*   npm أو yarn

### خطوات التشغيل
1. قم بتحميل المشروع.
2. قم بتثبيت المكتبات:
   ```bash
   npm install
   ```
3. ابدأ تشغيل خادم التطوير:
   ```bash
   npm run dev
   ```
4. لبناء نسخة الإنتاج:
   ```bash
   npm run build
   ```
